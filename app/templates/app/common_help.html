Список требований к проекту

  • Папка с изображениями называется image
  • Все пути к файлам заключены в двойные кавычки
  • Папка с файлами стилей называется css
  • Папка со скриптами называется js
  • Обязательно наличие файла index.html
  • Основной файл стилей должен называться style.css
  • Объекты для распознавания системой задаются списком ID

Ключевые ID задаются по определенным правилам:

  • Префикс page обозначает принадлежность элемента к общему содержанию страницы
  • Через дефис ставится название элемента

Пример задания ID для элемента: <header id="page-header"></header>

Примеры других ID: #page-header, #page-footer, #page-nav, #page-content
