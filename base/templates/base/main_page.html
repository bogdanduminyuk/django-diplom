Основное назначение этого сервиса - помочь Web-разработчику создать готовый шаблон сайта, который можно впоследствии установить на систему управления содержимым.

Входные данные - HTML файлы, результат - архив с шаблоном.

Ускорение разработки будет достигаться за счет различных готовых инструментов, которые объединяет этот сервис, позволяя преобразовать результаты их работы в единый архив.

Адаптация под CMS осуществляется при помощи плагинов (модулей), который может быть написан под любую CMS, включая различные версии одной и той же CMS.
